Imagens dinamicas

Dicas importantes:

  1. A tag IMG puxa a imagem que você quer mostrar no seu site e ela deve ser coloca na parte mais inferior do código dentro do PICTURE
  2. Utilizando a tag SOURCE:MEDIA:TYPE é possível carregar uma imagem e especificar em qual largura de tela ela deve ser carregada
  3. Colocar max-width no lugar de min-width, assim é possível selecionar uma imagem que se encaixe corretamente na largura da tela
  4. Para funcionar corretamente deve-se colocar em primeiro lugar a imagem de menor tamanho até a de maior tamanho

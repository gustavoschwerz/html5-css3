Código-fonte / Pré-formatação

Comparando duas formas de escrever um pequeno código, com e sem a tag CODE:

O comando documente.getElementByld('teste') é scrito em linguagem JavaScript. (sem a tag)

O comando documente.getElementByld('teste') é scrito em linguagem JavaScript. (com a tag)

Escrevendo um código em Phyton, usando a tag CODE + tag PRE:

        
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é ÍMPAR')
print('Fim do programa')

A tag PRE mantém exatamente a mesma formatação deixada durante a elaboração do código.
